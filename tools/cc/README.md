# cc — автоименование сессий Claude Code

Обёртка для `claude`, которая автоматически генерирует короткое название сессии из промпта. Название отображается во вкладке терминала с индикатором активности (`*` / `.`).

## Как это выглядит

```
$ cc сделай обзор рынка МФО агрегаторов в России, кто имеет витрины, какой оборот у юрлиц
```

Скрипт саммаризирует промпт через haiku (~3 сек) → запускает сессию с названием **«Обзор рынка МФО»** → вкладка терминала показывает `* Обзор рынка МФО` вместо `* Claude Code`.

## Установка

### 1. Настройка Cursor / VS Code

Добавь в settings.json (`Cmd+Shift+P` → `Preferences: Open User Settings (JSON)`):

```json
"terminal.integrated.tabs.title": "${sequence}"
```

### 2. Установка скрипта

```bash
# Скачать
curl -o ~/.local/bin/cc https://raw.githubusercontent.com/engwatch/insapp-skills/main/tools/cc/cc

# Сделать исполняемым
chmod +x ~/.local/bin/cc
```

### 3. Убедиться что `~/.local/bin` в PATH

```bash
# Проверить
echo $PATH | tr ':' '\n' | grep local/bin

# Если нет — добавить в ~/.zshrc (или ~/.bashrc):
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

## Использование

```bash
# Промпт аргументом — название генерится автоматически
cc сделай обзор рынка МФО агрегаторов в России

# Интерактивный режим — вводишь промпт после ›
cc

# Все флаги claude работают как обычно
cc --resume          # продолжить сессию
cc -c                # продолжить последнюю
cc --help            # справка claude
```

## Как это работает

1. `cc` получает промпт (аргументом или интерактивно)
2. Отправляет промпт в `claude -p --model haiku` с инструкцией «придумай короткое название»
3. Запускает `claude --name "Название" "промпт"` — флаг `--name` задаёт имя сессии
4. Claude Code отображает это имя во вкладке терминала (вместо «Claude Code»)

Саммаризация через haiku занимает ~3 секунды и стоит доли цента.

## Требования

- Claude Code CLI (`claude`)
- Python 3 (для обработки текста)
- Cursor / VS Code с настройкой `"terminal.integrated.tabs.title": "${sequence}"`

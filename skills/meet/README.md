# Meet — создание встреч

Создаёт встречу в Яндекс Телемост, находит участников по имени в таблице сотрудников, ставит событие в Google Календарь с приглашениями.

## Команда

```
/meet
Создай встречу на пятницу в 10:00, добавь Иванова и Петрову
Встреча завтра в 15:30 с ivan@gmail.com
```

## Что делает

1. Создаёт встречу в Яндекс Телемост
2. Находит Gmail участников в таблице сотрудников (если указано имя)
3. Создаёт событие в Google Календарь с ссылкой и приглашениями

## Требования

- `gdrive` MCP (Google Календарь + Таблицы)
- `telemost` MCP (Яндекс Телемост)

Полная инструкция по настройке: [SETUP.md](SETUP.md)

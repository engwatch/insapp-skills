# Publick Push — публикация скиллов в общий репозиторий

Синхронизирует скиллы из `~/.claude/skills/` в публичный репо `engwatch/insapp-skills`.

## Команда

```
/publick_push
```

## Что делает

1. Pull последнюю версию из GitHub/GitLab
2. Копирует обновлённые скиллы из `~/.claude/skills/`
3. Обновляет README при добавлении новых скиллов
4. Коммит и push

## Требования

- Git, SSH доступ к репозиторию

# GitLab Find Dev Repos — поиск всех репозиториев разработчика

Находит все проекты в GitLab, в которых работает конкретный разработчик — через MR и коммиты.

## Использование

Навык-справочник. Применяется автоматически когда нужно найти все project_id для разработчика.

## Как работает

1. Глобальный поиск через `list_merge_requests(scope="all", author_username=...)` — находит проекты с MR
2. Перебор project_id для `list_commits(author=...)` — находит проекты с коммитами без MR
3. Сбор статистики с `with_stats=true`

## Требования

- gitlab MCP

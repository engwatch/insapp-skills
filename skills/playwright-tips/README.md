# Playwright Tips — блокировка JS-редиректов и anti-DDoS

Справочник приёмов работы с Playwright: блокировка JS-редиректов на CPA-сайтах, обход anti-DDoS защит, альтернативные источники данных.

## Использование

Навык-справочник. Применяется автоматически при работе с защищёнными сайтами через Playwright.

## Основные приёмы

- Блокировка JS-редиректов через `page.route('**/*', ...)`
- Известные защиты: Qrator (vbr.ru), JS-rendered (partnerkin.com), CPA-редиректы
- Альтернативные источники: vc.ru, gdetraffic.com, zorbasmedia.ru

## Требования

- Playwright

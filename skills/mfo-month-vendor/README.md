# MFO Month Vendor — сверка МФО по вендорам за месяц

Создаёт Google Spreadsheet со сверкой по вендорам: листы по каждому партнёру + лист Итог с формулами.

## Команда

```
/mfo-month-vendor [партнёры] [период]
/mfo-month-vendor "Локо, Долинск" "февраль 2026"
```

## Что делает

1. Читает вендорские файлы, определяет партнёра по кампании
2. Копирует шаблон таблицы
3. Заполняет листы партнёров (A-H: дата, лид, ID, МФО, КВ, %, КВ партнёру, вендор)
4. Заполняет лист Итог с SUMPRODUCT формулами
5. Форматирование + тройная верификация

## Требования

- insapp-db MCP
- gdrive MCP
- Playwright (авто-ширина, таблица)

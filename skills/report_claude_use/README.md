# Report Claude Use — отчёт по использованию Claude Code

Анализирует все сессии Claude Code за указанный период, группирует по направлениям работы и генерирует HTML-дашборд.

## Команда

```
/report_claude_use
/report_claude_use последние 7 дней
/report_claude_use эта неделя
```

## Что делает

1. Сканирует JSONL-файлы сессий в `~/.claude/projects/`
2. Извлекает user messages, assistant texts, используемые инструменты
3. Категоризирует каждую сессию по ключевым словам
4. Формирует краткие итоги по каждой категории
5. Генерирует HTML-дашборд: KPI, карточки активностей, MCP-интеграции, хронология
6. Открывает в браузере и предлагает опубликовать через `/html-push`

## Структура отчёта

- Шапка с KPI (сессии, направления, MCP-сервисы, дни)
- Карточки по категориям (2 колонки) с итогами и датами
- Блок MCP-интеграций с описаниями
- Компактная хронология всех сессий

## Требования

Нет (работает с локальными файлами сессий Claude Code)

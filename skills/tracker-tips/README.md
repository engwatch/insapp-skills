# Tracker Tips — резолв аккаунтов Яндекс Трекера

Справочник по работе с Яндекс Трекер MCP: какие логины для каких инструментов, маппинг старых/новых аккаунтов.

## Использование

Навык-справочник. Применяется при работе с Трекером — помогает избежать ошибок при резолве пользователей.

## Ключевые факты

- `resolve_user` — русское имя ("Климов"), не транслит
- `get_employee_stats` — старый логин Яндекс-паспорта
- `search_issues` — новый логин

## Требования

- tracker MCP

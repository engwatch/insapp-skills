# HH Resume Search — поиск кандидатов на hh.ru

Эффективный поиск специалистов на hh.ru через Playwright. Обходит ограничения поиска hh.ru — short queries, form-based search, блок «похожие резюме».

## Использование

Навык-справочник. Применяется при поиске кандидатов через Playwright.

## Ключевые приёмы

- **Короткие запросы** (2-3 слова) дают более точные результаты, чем длинные
- **Form-based поиск** через `page.goto('/search/resume')` — URL-параметры ломают фильтры
- **Блок «Похожие резюме»** внизу каждого резюме — ML-матчинг, золотая жила

## Требования

- Playwright

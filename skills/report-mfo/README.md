# Report MFO — отчёт по МФО-партнёру

Создаёт отчёт по МФО-партнёру: визиты, переходы, выдачи, комиссии — по дням.

## Команда

```
/report-mfo "ЛОКО-БАНК" "6–10 марта 2026"
/report-mfo "Тинькофф" "март 2026"
```

## Что делает

1. Находит партнёра в БД Insapp
2. Запрашивает статистику по дням (визиты, переходы в МФО, выдачи, комиссии)
3. Выводит таблицу в терминале
4. Если настроен gdrive MCP — создаёт Google Sheet по шаблону

## Пример вывода

```
Отчёт: ЛОКО-БАНК | 6–8 марта 2026

| Дата       | Переходов | МФО           | Выдачи | Вход. КВ | Исх. КВ  | Доход   |  CR   |
|------------|-----------|---------------|--------|----------|----------|---------|-------|
| 06.03.2026 |       229 | OneClickMoney |      1 |  9 000 ₽ |  7 200 ₽ | 1 800 ₽ | 0.44% |
| 07.03.2026 |       169 | OneClickMoney |      2 | 18 000 ₽ | 14 400 ₽ | 3 600 ₽ | 1.18% |
| ИТОГО      |       398 | —             |      3 | 27 000 ₽ | 21 600 ₽ | 5 400 ₽ | 0.75% |
```

## Требования

- `insapp-db` MCP (обязательно)
- `gdrive` MCP (опционально, для Google Sheets)
